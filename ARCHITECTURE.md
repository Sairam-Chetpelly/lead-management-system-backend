```

   POST /api/auth/login
   { email, password }
        в”Ӯ
        в–ј
2. Middleware Chain
   CORS вҶ’ Rate Limiter вҶ’ API Key вҶ’ Validation
        в”Ӯ
        в–ј
3. Route Handler
   auth.routes.js
   POST /login вҶ’ validateLogin вҶ’ authController.login
        в”Ӯ
        в–ј
4. Controller
   auth.controller.js
   - Validate credentials
   - Generate JWT token
   - Return response
        в”Ӯ
        в–ј
5. Model Layer
   User.findOne({ email })
   user.comparePassword(password)
        в”Ӯ
        в–ј
6. Database
   MongoDB query
   Return user document
        в”Ӯ
        в–ј
7. Response
   { token, user, expiresAt }
```
